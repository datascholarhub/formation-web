# Formation Web

Ce dépôt contient les supports de cours et les exercices de la formation web.

## Contenu du dépôt

- `cours/` : les fichiers de démonstration et les exemples vus en séance.
- `exercices/` : les consignes, contenus fournis et rendus attendus pour la pratique.


## Organisation des fichiers

L’arborescence principale est la suivante :

```text
README.md
cours/
	seance1/
    seance2/
    ...
exercices/
	seance1/
	seance2/
    ...
solutions/
    seance1/
    seance2/
    ...
```
Chaque séance a son propre dossier dans les répertoires `cours/`, `exercices/` et `solutions/` pour faciliter la navigation et l’organisation des ressources.

## Extensions recommandées pour Visual Studio Code :
- **Auto Rename Tag:** permet de renommer automatiquement les balises HTML correspondantes.
- **Auto Close Tag:** ferme automatiquement les balises HTML.
- **Live Server:** lance un serveur local pour prévisualiser les pages HTML en temps réel.
- **Prettier - Code formatter:** formate automatiquement le code HTML/CSS/JS pour une meilleure lisibilité.
- **HTML CSS Support:** offre une meilleure prise en charge de l’autocomplétion et de la validation pour les fichiers HTML et CSS.
- **W3C Web Validation:** valide le code HTML selon les standards du W3C.

## Extensions recommandées pour les navigateurs :
Tout d'abord, il est important d’avoir un navigateur moderne et à jour pour bénéficier des dernières fonctionnalités de développement web. Voici quelques extensions utiles pour les développeurs web :
- **ColorZilla:** un outil de sélection de couleurs pour extraire les couleurs d’une page web.
- **WhatFont:** permet d’identifier les polices utilisées sur une page web.
